# 微信小程序－网盘搜索

### 说明：

实现疾病搜索功能，特色：
- 使用fontawesome字体

### 数据接口:

- https://api.getweapp.com/vendor/zainot/GetArticleList
- https://api.getweapp.com/vendor/zainot/GetArticleById
- https://api.getweapp.com/vendor/zainot/GetDieaseList

### 目录结构：

- icons — 存放项目图片文件
- pages — 存放项目页面文件
- utils — 存放格式化文件
- fonts — 存放字体文件

### 开发环境：

微信web开发者工具 v0.11.122100
